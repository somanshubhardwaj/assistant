# ASSISTANT

Assistant is a python project which convert speech from one language to other.

## Modules used

- speech_recognition
- googletrans
- gtts
- os
- Pyaudio

> Project is made to be only functional on  windows


# Module installation

```python
pip install SpeechRecognition
pip install googletrans==4.0.0-rc1
pip install gTTS
pip install pyaudio
```
